# 🌱 [Focus Garden](https://scostadavid.github.io/focusgarden/)

> Productivity focused app pomodoro + todolist written in sveltejs, bundled with vite


## Installation

```bash
$ yarn install
```

## Running the app
```bash
# development
$ yarn dev

# build static pages for deploy
$ yarn build
```
